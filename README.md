## Beijing-Secondary-School-AI-Contest-2025

This repository is maintained by secondary-school teachers and collects contest-style exercises, reference solutions, and environment setup instructions inspired by the Beijing Secondary School AI Competition.
It is intended solely for teaching and training purposes and is not an official repository of the competition organizers.

#### Step-1 : Environment Setup

* install requirements

```shell
pip install -r requirements.txt
```

* create jupyter kernel

```shell
python -m ipykernel install --user --name=beijing-secondary-school-ai-contest-2025
```

#### Step-2 : unzip glove.6B.50d model

* 解压缩 glove.6B.50d.txt.zip 模型

```shell
unzip glove.6B.50d.txt.zip
```
